# Crill-IOS – VS Code Plugin für Cisco IOS Konfigurationen

Crill-IOS ist eine VS Code-Erweiterung, die dich beim Schreiben von Cisco IOS Konfigurationen unterstützt – mit kontextsensitiver Autovervollständigung, Syntax-Highlighting und Validierung.

## Quickstart
1. Installiere die Erweiterung in VS Code. Weiter geht’s bei der [Installation](./installation.md).
2. Erstelle oder öffne eine Datei mit der Endung `.ios`.
3. Tippe z. B. `configure` oder zeige Vorschläge an mit `Ctrl+Space`.

## Was du bekommst
- **Autovervollständigung**: Vorschläge passen sich dem jeweiligen Modus (z. B. `configure terminal`, `interface...`, `router...`) an.
- **Syntax-Highlighting**: Befehle, Parameter und Kommentare werden übersichtlich eingefärbt.
- **Validierung & Hinweise**: Fehler/Warnungen erscheinen in VS Codes „Problems“-Ansicht; per Hover bekommst du Details.
- **Best Practices (Hinweise)**: Das Plugin hilft typische Fehler schnell zu erkennen.
