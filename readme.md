# Installation / Setup

First create a new folder and clone the repo Crill-IOS-Docs.

## Setup virtual environment for python
- `python3 -m venv .venv`
- `source .venv/bin/activate`

## Install python packages
- `pip install -r requirements.txt`

# Run the local server
- `cd Crill-IOS`
- `mkdocs serve`

# Deploy
- `cd Crill-IOS`
- `mkdocs gh-deploy`
